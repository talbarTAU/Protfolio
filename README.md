Retro Portfolio 🎮 - Tal Bar

About the Project

An interactive personal portfolio website designed entirely in a retro / Pixel Art (8-bit) style. The site was built as part of Assignment 2 for a course at Shenkar, aiming to reflect my unique professional identity: a combination of a software engineering background (BSc in Computer Science at TAU), a Master's degree in Game Design and Development at Shenkar, and my current role as a Game Economist at SuperPlay.

The design draws inspiration from the gaming world and includes:

Character Stats: The CV (resume) section is designed like a game character's status screen (Skill Tree, Quest History).

Retro UI: Usage of pixel fonts, thick borders, hard shadows, and floating animations (Floating XP).

Cohesive Visual Language: The colors are based on SuperPlay's color palette (vibrant pink and orange).

Main Technologies:

HTML5: Semantic structure of the site.

Tailwind CSS: For fast and responsive styling (loaded via CDN).

CSS Custom Styles: Animations (Keyframes), smooth scrolling, and precise Pixel-Art settings.

Google Fonts: Using Press Start 2P and VT323 fonts to achieve the authentic 8-bit look.

How to Run the Project

The website is a purely client-side static site (HTML/CSS), so there is no need for special installations (like Node.js) or a complex local server.

Option 1: Direct Browser Opening (The Simplest Way)

Download the repository as a ZIP file (or run git clone).

Extract the files to a folder on your computer.

Ensure all media files (the profile picture Gemini_Generated_Image_8l91th8l91th8l91.png, project images, the ekko_procreate.mp4 video, and the Tal_Bar_CV_MDES.pdf file) are in the same folder along with the index.html file.

Simply double-click on the index.html file. The website will open in your default browser and run fully.

Option 2: Running via Live Server (Recommended for Developers/Reviewers)

If you want to view the site running through a local development server (recommended to prevent browser security blocks on loading local videos or fonts in some cases):

Open the project folder in a code editor, such as Visual Studio Code.

Ensure you have the Live Server extension installed.

Right-click on the index.html file in the file explorer sidebar.

Select the "Open with Live Server" option. The website will automatically open in the browser at the local address (usually http://127.0.0.1:5500).
